# Portfolio

## Project Overview
Personal portfolio website built with React + Vite + TypeScript.

## Tech Stack
- **Framework**: React 19 + Vite 8 + TypeScript
- **Styling**: CSS (no framework specified yet)
- **Deployment**: GitHub Pages

## Folder Structure
```
src/
├── assets/
│   ├── images/      # photos, project screenshots
│   ├── icons/       # social icons, skill icons
│   └── resume/      # resume PDF
└── components/      # reusable React components
```

## Getting Started
```bash
npm install
npm run dev     # development server at http://localhost:5173
npm run build  # production build
```

## Notes
- Assets in `src/assets/` are imported directly into components
- Static assets that need a specific URL path go in `public/`
